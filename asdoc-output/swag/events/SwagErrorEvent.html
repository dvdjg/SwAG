Implements swag.interfaces.events.ISwagErrorEvent

Extends the SwagEvent class for broadcasting error messages generated by the SwAG Toolkit. This class is used to broadcast any errors that are not thrown (i.e. non-fatal code errors).

public function SwagErrorEvent(type:String = null, bubbles:Boolean = false, cancelable:Boolean = false)

Creates a new SwagErrorEvent instance which can subsequently be broadcast using the standard Flash event system.

Parameters
type:String (default = null) — The type of event to broadcast. If this parameter is omitted, or is null, or a blank string, a generic SwagErrorEvent.ERROR-type event will be created. Use the included static properties for other error codes.
 
bubbles:Boolean (default = false) — Passed as is to the standard Flash Event constructor.
 
cancelable:Boolean (default = false) — Passed as is to the standard Flash Event constructor.
