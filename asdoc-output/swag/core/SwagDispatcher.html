Classpublic class SwagDispatcher
InheritanceSwagDispatcher Object
Implements swag.interfaces.core.ISwagDispatcher
Subclasses SwagDataTools, SwagSystem

Provides global event broadcasting capabilities similar to the standard Flash event system.

Unlike Flash events, the SwAG event system is decoupled by default (event listeners are not required to be bound to specific dispatchers), thereby providing a much greater degree of flexibility. For example, listeners may be established before the dispatching object exists, which means that an application can be assembled at runtime without the need for any specific load order. Furthermore, decoupled events allow the application to dispatch from / listen to static objects, something that is not supported with the standard event system.

addEventListener()method
public static function addEventListener(eventType:String, eventMethod:Function, thisRef:* = null, sourceObject:* = null):SwagEventListener

Adds an event listener to the SwagDispatcher.

The format of the listener is very similar to a standard Flash event but, because SwagDispatcher events can be decoupled, it also provides an optional sourceObject property for event filtering based on the dispatcher, as well as a reference to the containing object (thisRef) which the dispatcher uses to intelligently invoke listening methods (ones that don't necessarily match the required format -- a SwagEvent-type object as the first parameter).

Parameters

eventType:String — The event type to broadcast. While this may be a standard string, it's advisable to use the various event string constants provided with the event objects in order to prevent having to implement numerous changes if event strings ever change.
 
eventMethod:Function — The method to invoke when the eventType is dispatched.
 
thisRef:* (default = null) — A reference to the object (class, instance, etc.), containing the method to be invoked. If this parameter is not supplied, or is null, the event is unable to verify that the listening method has the correct parameters and this may result in runtime errors (i.e. you must be careful to include a SwagEvent or related type as the first parameter). If thisRef is supplied and contains the target method, the event broadcaster attempts to invoke the event method more intelligently (i.e. the dispatcher will attempt to invoke the method with the correct parameter types, even if they don't match the standard event listener method format).
 
sourceObject:* (default = null) — The object(s) from which the event is expected. If this is null, any object will be considered a match for the event, otherwise only the specified object(s) will be a match and other events, even if they have the same eventType, won't invoke the eventMethod. This parameter may either be a singular object or an array of object references.

Returns
SwagEventListener — The newly created SwagEventListener instance created, or null if there was a problem creating the listener.

dispatchEvent()method 
public static function dispatchEvent(eventObj:ISwagEvent, source:*):Boolean

Dispatches a standard, de-coupled SwagEvent-type event.

Like standard Flash events, SwAG events are matched according to their type property. Unlike standard Flash events, however, SwAG events are not bubbled or propagated through the class inheritance chain. Rather, they are broadcast on a first-come, first-serve basis (in the order in which they're added), anywhere throughout the application.

Parameters

eventObj:ISwagEvent — A SwagEvent instance, extending instance, or implementation of ISwagEvent
 
source:* — The source object (the object / class / instance / etc.) from which the event is being dispatched. Listeners that are not bound to a specific dispatcher will be invoked regardless of this parameter, but methods bound to only a specific dispatcher use this value to determine whether or not they should be invoked.

Returns

removeEventListener()method 
public static function removeEventListener(eventType:String, eventMethod:Function, sourceObject:* = null):Boolean

Removes an event listener created with the addEventListener method.

Parameters

eventType:String — The event type to remove. This, along with the eventMethod and, optionally, the sourceObject, are used to determine which listener(s) should be removed.
 
eventMethod:Function — The method associated with the event to remove.
 
sourceObject:* (default = null) — The optional source object associated with the event to remove. If this is null, this parameter is essentially ignored. If this is a singular object reference, it must match the object used to register the event with the addEventListener call. If this object is an array, any of the objects in the array are considered a source match for removal.

Returns
Boolean — True is the listener was successfully removed, false if there was a problem or no associated listener could be found to remove.

removeOrphanedListeners()method 
public static function removeOrphanedListeners():void

Removes any orphaned event listeners created by the SwagDispatcher. Orphaned listeners are those that still exit in memory but whose type or listening methods are null (no longer exist). Removing orphaned listeners helps to maintain application memory by removing any references which will no longer be used.
